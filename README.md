# Scroll-Insta 🚀

Turned from Script → To Executable.

Scroll-Insta is a simple Instagram automation tool packaged as a Windows executable (.exe).  
No coding knowledge required — just download and run.

---

## 🔥 Features

- Automated Instagram scrolling
- Profile session handling
- Dynamic DOM interaction
- Packaged as a standalone Windows application
- No Python installation required

---

## 🛠 Built With

- Python
- Selenium
- PyInstaller

---

## 📦 Installation & Usage

1. Go to the repository:
   https://github.com/Ajay-308/scroll-insta-exe/tree/main

2. Download the `.exe` file.

3. Run the application.

That's it.  
No setup. No dependencies. Just install and use.

---

## 💡 Use Case

You can keep Instagram scrolling in the background while you:
- Code
- Study
- Work on other tasks

Automate the scroll. Focus on what matters.
